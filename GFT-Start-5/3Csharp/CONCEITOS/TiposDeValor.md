- Variáveis de tipo de valor contêm diretamente sues dados.
- Numéricos
  - sbyte
  - short
  - int
  - long
  - byte
  - ushort
  - uint
- Caractere unicode
  - char
- Pontos Fluntuantes
  - float
  - double
  - decimal
- Booleana
  - bool
- Outros
  - enum
  - struct 
  - nullable
